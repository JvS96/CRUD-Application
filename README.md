# ProPay

A web-based system where a user can log in and out of the system to manage people via
CRUD operations.

Tech Stack:
-PHP (PHPStorm) 7.4.9
-MySQL 8.0.21
-Laragon/Wamp64
-JavaScript
-HTML and CSS
-Bootstrap
